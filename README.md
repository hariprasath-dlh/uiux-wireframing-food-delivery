# UI/UX Wireframing – Food Delivery App

This repository contains a small UI/UX case study focused on
**wireframing and user flow design**.

## Objective
To understand how low-fidelity wireframes and user flows help in
planning screen structure and improving usability before final UI design.

## User Flow
Login → Home → Restaurant Menu → Cart / Checkout → Order Confirmation

## Work Included
- Low-fidelity wireframes (mobile)
- User flow definition
- Basic UX reasoning

## Tools Used
- Figma

## Note
This is a learning project created as part of a UI/UX internship
assignment.
